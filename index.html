How the estimate works: every time the official count goes up, that's a real arrival. We assume each person's visit length varies — most stay around an hour, some leave sooner, some later — so instead of waiting for the source to report an exit, we gradually "fade out" each arrival over time based on how long visits typically last. The % you see is that estimated number of people still likely present, divided by the highest we've realistically seen recently (not the building's official max capacity).
